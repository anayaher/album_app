### Installation

1. Clone the repository:

```bash
   git clone https://github.com/anayaher/flutter_news_app.git
```
2. Navigate to the project directory::

```bash
   cd flutter_news_app
```
3. Install dependencies :

```bash
  flutter pub get
```
3.  Run app:
```bash
    flutter run
```
